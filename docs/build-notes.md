# 制作与烧录说明

## 制作要点

1. 焊接完成后，请测量电池正极到电阻一侧的电压。
2. 如果发现电压缓慢下降或者无电压，通常是电池底部接触不良，需要在底部焊上一点锡顶住电池。
3. `Plus` 版排阻中，最左侧一颗为 `30Ω`，右侧七颗为 `470Ω`。

## 固件与硬件版本匹配

- `v1.1` 固件适配硬件 `v1.1 / v1.2 / v1.21`
- `v1.21` 固件仅适用于硬件 `v1.21 / v1.22`

## NDS 烧录

根据原项目说明，配合 NDS 上的 `gbcburn` 烧录软件，可以实现使用 NDS 对此卡进行烧录，从而减少额外购买专用烧录器的需求。

## 推荐制作流程

1. 选择硬件版本
2. 下单 PCB 或直接使用仓库中的 Gerber
3. 完成焊接并检查电池与关键阻值
4. 烧录匹配版本的 POF 固件
5. 如果使用多合一卡方案，再处理菜单 ROM 与配置文件
6. 上机进行功能验证
