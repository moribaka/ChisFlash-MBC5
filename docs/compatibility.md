# 兼容性与功能现状

## 当前公开说明

- 项目主要面向 `MBC5` 游戏
- 理论兼容部分 `MBC1 / MBC2 / MBC3` 游戏
- 兼容性目前并未在仓库中完整量化列出
- `Plus` 版使用电平转换芯片，普通版对 AP 掌机的兼容情况仍待更多测试

## 多合一卡支持

- `8M` 版本支持 `单卡`
- `8M` 版本支持 `3合1 (1+2+4)`
- `8M` 版本支持 `4合1 (1+2+2+2)`
- `MAX` 版本支持 `16合1`

## 未来方向

- 支持震动
- 支持更多合卡组合
- 补充更完整的兼容性实测结果

## 仓库层面的缺口

目前仓库里已经有源码和生产文件，但还缺少这些会让项目更完整的内容：

- 兼容游戏列表
- 已验证平台与掌机列表
- 烧录器 / 烧录流程的详细截图
- 成品板实拍与装配要点图
